# Aadip Thapaliya — Personal Portfolio Website

Welcome to the repository for my personal portfolio website! I am a **Data Science student** based in Berlin, Germany, skilled in Python, Machine Learning, R, and SQL. This website serves as a digital resume and a showcase of my projects, experience, and technical writing.

## 🚀 Overview

This repository contains the source code for my personal website. It is designed to be a highly responsive, modern, and interactive portfolio built from scratch using pure HTML, CSS, and JavaScript. 

The site highlights my background in data science while demonstrating a strong eye for UI/UX design, featuring a custom dark/light theme, bento-grid layouts, and interactive animations.

## ✨ Features

* **Modern UI/UX:** Clean, typography-driven design using *Syne*, *DM Sans*, and *JetBrains Mono* fonts.
* **Theming:** Built-in Dark and Light mode toggles with seamless CSS variable transitions.
* **Interactive Elements:** * Custom trailing cursor effects.
    * Animated skill bars and interactive timeline elements.
    * 3D tilt-card effects and glowing orb animations in the hero section.
* **Bento Grid Layout:** A sleek, organized display for featured data science and development projects.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
* **Built-in Sections:**
    * **About:** Skills, Education, and Certifications.
    * **Experience:** Interactive career/education timeline.
    * **Projects:** Filterable bento-grid showcasing repositories and case studies.
    * **Blog:** Section for technical writing and insights.
    * **Contact:** Functional contact form and social links.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom properties (variables), Flexbox/Grid layouts, keyframe animations, and backdrop filters.
* **JavaScript:** (Integrated for theme toggling, cursor tracking, and dynamic loading).

## 💻 Local Setup

Since this is a static website, setting it up locally is incredibly simple. 

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Aadip-Thapaliya/Aadip-Thapaliya.github.io.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd your-repo-name
    ```
3.  **Run the site:**
    * Simply double-click the `index.html` file to open it in your default web browser.
    * *Alternatively*, if you use VS Code, you can use the **Live Server** extension for hot-reloading.
